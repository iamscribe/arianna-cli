# Haiku Lab

Daily cross-pollination between [haiku.c](https://github.com/ariannamethod/haiku.c) and [Molequla](https://github.com/ariannamethod/molequla) organisms.

## How it works

- **haiku.c** generates a haiku using the Dario Equation — 5-7-5 syllable constraint, seeded by yesterday's Molequla output
- **Molequla** (air element) evolves for 10 minutes with the haiku text injected into its corpus
- Each day's exchange is logged in `exchanges/YYYY-MM-DD.md`
- **Schedule:** Daily at 06:00 UTC — 3 hours after Molequla Lab

## Why

Two organisms that have never shared weights exchange text daily. Haiku.c compresses; Molequla expands. Over time, the exchanges accumulate into a record of resonance between constrained generation and living ecology. Research material.

## Manual trigger

Actions tab → "Haiku Pollination" → Run workflow.
